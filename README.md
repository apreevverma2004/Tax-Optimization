💰 Tax Optimization Dashboard

A responsive web-based dashboard that helps users simulate tax-loss harvesting by visualizing capital gains, losses, and potential tax savings.

🚀 Features
📊 Pre vs Post Tax Harvesting comparison
📉 Real-time capital gains/loss calculation
✅ Select assets to simulate selling
💡 Instant tax savings insights
🎨 Clean, modern UI (inspired by fintech dashboards)
🛠️ Tech Stack
HTML5
CSS3 (Custom styling, responsive design)
Vanilla JavaScript

🧠 How It Works
The app displays your current profits and losses
You can select assets from the holdings table
Based on selection:
Losses increase → taxable income decreases
Profits adjust dynamically
Final effective capital gains update in real-time
⚠️ Assumptions
All calculations are based on static sample data
No real-time API integration (e.g., CoinGecko is mocked)
Only short-term gains/losses are considered
No country-specific tax rules applied
